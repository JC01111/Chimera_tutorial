### Select command
#### Set the background color to be white
    set bgColor white
#### Select All
    select all
#### Select Chain A (/A/B to select multiple chains)
    select /A   
#### Clear selection
    select clear
    
### Hide command    
#### Hide selected atoms
    hide sel atoms
#### Hide selected cartoons
    hide sel cartoons

### Color command
#### color all to be specific color
    color white
#### color specific chain to be a color
    color /A white
